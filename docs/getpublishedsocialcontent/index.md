---
title: "GetPublishedSocialContent"
---


This function returns previous Social Content specified by the Region ID.

### Argument

`GetPublishedSocialContent(1)`

| Ordinal | Type | Required | Description |
| --- | --- | --- | --- |
| 1 | Number | True | Region ID of the Social Content Area to return |

> NOTE: This function is not supported in the current version of Marketing Cloud.
>
> NOTE: This function is only for use in Microsites and Landing Pages and will only return Content Areas built in Classic Content.

### Example

```html
%%=GetPublishedSocialContent(12345)=%%
```
