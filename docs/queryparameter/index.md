---
title: "QueryParameter"
---


Returns the value of a parameter from a form field or landing page URL parameter. This function can also retrieve parameters from an encrypted query string generated by the `CloudPagesURL()` function.

### Arguments

| Ordinal | Type   | Required | Description                                                              |
|---------|--------|----------|--------------------------------------------------------------------------|
| 1       | String | Yes      | The name of the URL query parameter to retrieve its corresponding value. |

> NOTE: This function behaves the same way as the `RequestParameter()` function. Both functions are retained to preserve backward compatibility.

### Example 1: Basic Use

```html
/* Define a default value for the 'name' parameter */
VAR @name
SET @name = QueryParameter('name')

/* Check if the parameter has a value and display it */
IF NOT EMPTY(@name) THEN
    OutputLine(Concat("Hello, ", @name, "!"))
ELSE
    OutputLine("Hello, Guest!")
ENDIF
```

### Example 2: Advanced Scenario

```html
/* Define variables for multiple parameters */
VAR @firstName, @itemName, @campaignId
SET @firstName = QueryParameter('firstName')
SET @itemName = QueryParameter('itemName')
SET @campaignId = QueryParameter('campaignId')

/* Output personalized content based on retrieved parameters */
IF NOT EMPTY(@firstName) THEN
    OutputLine(Concat("Dear ", @firstName, ","))
ELSE
    OutputLine("Dear Customer,")
ENDIF

IF NOT EMPTY(@itemName) THEN
    OutputLine(Concat("Your selected item is: ", @itemName, "."))
ENDIF

IF NOT EMPTY(@campaignId) THEN
    OutputLine(Concat("This message is part of campaign: ", @campaignId, "."))
ENDIF

/* Example of how the URL might look for this scenario:
   https://www.example.com/landingpage?firstName=John&itemName=Laptop&campaignId=12345
*/
```
