title: "BarCodeURL"

Generates a barcode image URL based on an input string. This function supports various linear (one-dimensional) and two-dimensional barcode types like Data Matrix and PDF417. It is important to note that this function **cannot** be used to generate QR codes.

> NOTE: The `BarCodeUrl()` function can be called up to two times per message or landing page.
> NOTE: This function **cannot** be used to create QR codes.
> NOTE: When using optional parameters, all preceding parameters must be included. If a value is not needed, an empty string can be passed.
> NOTE: Some barcode types have fixed character limits. Providing an incorrect number of characters for these types will prevent the barcode from rendering.
> NOTE: The following barcode types are not supported and will not produce an exception or output a barcode: `mat25`, `onecode`, `plaintext`, `planet`, `postnet`, `telepen`.

### Example 1: Basic Use

This example demonstrates how to generate a basic Code 128 Auto barcode with a specific value, width, and height. The barcode will display the text below it and have a white background.

```html
%%[
    /* Define the value to be encoded in the barcode */
    SET @barcodeValue = "ORDER12345ABC"

    /* Define barcode parameters */
    SET @barcodeType = "code128auto"
    SET @width = 200
    SET @height = 100
    SET @boolShowText = true /* Display the value under the barcode */
    SET @boolTransparentBG = false /* White background */

    /* Generate the barcode URL */
    IF NOT EMPTY(@barcodeValue) THEN
        SET @barcodeImageURL = BarCodeURL(
            @barcodeValue,
            @barcodeType,
            @width,
            @height,
            "", /* checksumValue - not needed for code128auto */
            @boolShowText,
            "", /* altText - not used when boolShowText is true */
            0,  /* rotation */
            @boolTransparentBG
        )
    ELSE
        SET @barcodeImageURL = ""
    ENDIF

    /* Output the barcode image in HTML */
    IF NOT EMPTY(@barcodeImageURL) THEN
]%%
        <img src="%%=v(@barcodeImageURL)=%%" alt="Barcode for Order: %%=v(@barcodeValue)=%%" style="width:%%=v(@width)=%%px; height:%%=v(@height)=%%px;">
%%[
    ENDIF
]%%
```

### Example 2: Advanced Scenario - QR-like Data Matrix with Custom Alt Text and Rotation

This example generates a Data Matrix barcode, which can encode more data than linear barcodes and is often used as a QR code alternative. It includes custom alt text, a transparent background, and a 90-degree rotation.

```html
%%[
    /* Define the value to be encoded in the barcode (e.g., a URL or complex ID) */
    SET @dataMatrixValue = "https://www.example.com/product/SKU789012345?promo=SUMMER2024"

    /* Define barcode parameters */
    SET @barcodeType = "datamatrix"
    SET @width = 150
    SET @height = 150
    SET @checksumValue = "" /* Not applicable for Data Matrix */
    SET @boolShowText = false /* Do not display the value under the barcode */
    SET @altText = "Scan for product details and promotion!"
    SET @rotation = 90 /* Rotate 90 degrees clockwise */
    SET @boolTransparentBG = true /* Transparent background */

    /* Generate the barcode URL */
    IF NOT EMPTY(@dataMatrixValue) THEN
        SET @dataMatrixImageURL = BarCodeURL(
            @dataMatrixValue,
            @barcodeType,
            @width,
            @height,
            @checksumValue,
            @boolShowText,
            @altText,
            @rotation,
            @boolTransparentBG
        )
    ELSE
        SET @dataMatrixImageURL = ""
    ENDIF

    /* Output the barcode image in HTML */
    IF NOT EMPTY(@dataMatrixImageURL) THEN
]%%
        <img src="%%=v(@dataMatrixImageURL)=%%" alt="%%=v(@altText)=%%" style="width:%%=v(@width)=%%px; height:%%=v(@height)=%%px; border: 1px solid #ccc;">
        <p style="font-size: 10px; text-align: center;">%%=v(@altText)=%%</p>
%%[
    ENDIF
]%%
```
