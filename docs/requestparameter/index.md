---
title: "RequestParameter"
---


The `RequestParameter()` function retrieves the value of a parameter from a form field or a landing page URL. It can also extract parameters from an encrypted query string generated by the `CloudPagesURL()` function. This function behaves identically to the `QueryParameter()` function, with both retained for backward compatibility.

### Arguments

| Ordinal | Type   | Required | Description                                     |
|---------|--------|----------|-------------------------------------------------|
| 1       | String | Yes      | The name of the query parameter to retrieve the value from. |

> NOTE: This function is commonly used in CloudPages or landing pages to dynamically personalize content based on URL parameters or form submissions.

### Example 1: Basic Use

This example demonstrates how to retrieve a `firstName` parameter from a URL and display a personalized greeting on a landing page.

```html
%%[VAR @firstName]
SET @firstName = RequestParameter("firstName")

/* Check if the firstName parameter is not empty before displaying */
IF NOT EMPTY(@firstName) THEN
    Output(Concat("Hello, ", @firstName, "!"))
ELSE
    Output("Hello there!")
ENDIF
]%%
```

### Example 2: Advanced Scenario with CloudPagesURL

This example shows how to use `RequestParameter()` to retrieve values from an encrypted query string generated by `CloudPagesURL()`, allowing for secure passing of subscriber data to a landing page.

```html
%%[
VAR @subscriberKey, @productName, @cloudPageURL

/* Define subscriber key and product name for the URL */
SET @subscriberKey = "12345"
SET @productName = "PremiumWidget"

/* Generate a CloudPagesURL with encrypted parameters */
SET @cloudPageURL = CloudPagesURL(123, "sk", @subscriberKey, "prod", @productName)

/* Output the generated URL (for demonstration purposes) */
/* Output(Concat("Click here: <a href=\"", @cloudPageURL, "\">Product Page</a>")) */

/* On the landing page (ID 123), retrieve the parameters */
VAR @retrievedSK, @retrievedProd
SET @retrievedSK = RequestParameter("sk")
SET @retrievedProd = RequestParameter("prod")

/* Check if retrieved values are not empty before displaying */
IF NOT EMPTY(@retrievedSK) THEN
    Output(Concat("Subscriber Key: ", @retrievedSK, "<br>"))
ENDIF

IF NOT EMPTY(@retrievedProd) THEN
    Output(Concat("Product Name: ", @retrievedProd))
ENDIF
]%%
```
