---
title: "String Functions"
---


String functions in AMPscript allow you to manipulate, format, and evaluate text data. These functions are essential for personalizing content, cleaning data, and building dynamic strings.

### Available Functions

| Function | Description |
| --- | --- |
| [Char](/char) | Returns the character for a specified ASCII value. |
| [Concat](/concat) | Concatenates two or more strings together. |
| [Format](/format) | Formats a value using a .NET format string. |
| [IndexOf](/indexof) | Returns the position of the first occurrence of a substring within a string. |
| [Length](/length) | Returns the number of characters in a string. |
| [Lowercase](/lowercase) | Converts a string to all lowercase characters. |
| [ProperCase](/propercase) | Converts a string to proper case (first letter of each word capitalized). |
| [RegExMatch](/regexmatch) | Returns a match from a string based on a regular expression pattern. |
| [Replace](/replace) | Replaces all occurrences of a specified substring within a string. |
| [ReplaceList](/replacelist) | Replaces a value with a corresponding value from a list. |
| [StringToDate](/stringtodate) | Converts a string representation of a date to a date value. |
| [StringToHex](/stringtohex) | Converts a string to its hexadecimal representation. |
| [Substring](/substring) | Returns a portion of a string starting at a specified position. |
| [Trim](/trim) | Removes leading and trailing whitespace from a string. |
| [Uppercase](/uppercase) | Converts a string to all uppercase characters. |
