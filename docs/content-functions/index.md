---
title: "Content Functions"
---


Content functions in AMPscript allow you to retrieve, include, and manipulate content within your emails, landing pages, and CloudPages. These functions are essential for building modular, reusable email templates.

### Available Functions

| Function | Description |
| --- | --- |
| [AttachFile](/attachfile) | Attaches a file to an outbound email message. |
| [BarcodeURL](/barcodeurl) | Returns a URL to a barcode image. |
| [BeginImpressionRegion](/beginimpressionregion) | Marks the beginning of an impression tracking region. |
| [BuildOptionList](/buildoptionlist) | Builds an HTML select option list from a data set. |
| [BuildRowSetFromString](/buildrowsetfromstring) | Creates a rowset from a delimited string. |
| [BuildRowSetFromXML](/buildrowsetfromxml) | Creates a rowset from XML data. |
| [ContentArea](/contentarea) | Returns the content of a Classic Content area by ID. |
| [ContentAreaByName](/contentareabyname) | Returns the content of a Classic Content area by name. |
| [ContentBlockByID](/contentblockbyid) | Returns the content of a Content Builder block by ID. |
| [ContentBlockByKey](/contentblockbykey) | Returns the content of a Content Builder block by Customer Key. |
| [ContentBlockByName](/contentblockbyname) | Returns the content of a Content Builder block by name. |
| [ContentImageByID](/contentimagebyid) | Returns an image tag for a Content Builder image by ID. |
| [ContentImageByKey](/contentimagebykey) | Returns an image tag for a Content Builder image by Customer Key. |
| [CreateSmsConversation](/createsmsconversation) | Starts an SMS conversation. |
| [EndImpressionRegion](/endimpressionregion) | Marks the end of an impression tracking region. |
| [EndSmsConversation](/endsmsconversation) | Ends an SMS conversation. |
| [GetPortfolioItem](/getportfolioitem) | Retrieves an item from the Classic Content portfolio. |
| [Image](/image) | Returns an image tag for a portfolio item. |
| [SetSmsConversationNextKeyword](/setsmsconversationnextkeyword) | Sets the next keyword in an SMS conversation flow. |
| [TransformXML](/transformxml) | Transforms XML content using an XSL stylesheet. |
| [TreatAsContent](/treatascontent) | Evaluates a string as AMPscript and HTML content. |
| [TreatAsContentArea](/treatascontentarea) | Evaluates a string as a Content Area. |

| [WATP](/watp) | Web Analytics Tracking with Parameters. |
