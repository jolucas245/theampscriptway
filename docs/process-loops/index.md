The `process-loops` concept in AMPscript primarily refers to the `FOR` loop construct, which is used to iterate through a block of code a specified number of times. It allows for controlled repetition, either incrementing or decrementing a counter variable within a defined range. This is fundamental for tasks requiring repetitive actions, such as processing data rows, generating sequential content, or iterating over collections.

### Example 1: Basic Use

This example demonstrates a simple `FOR` loop to display numbers from 1 to 5.

```html
%%[
    /* Define the counter variable */
    VAR @i

    /* Loop from 1 to 5 */
    FOR @i = 1 TO 5 DO
]
        <p>Current iteration: %%=v(@i)=%%</p>
%%[
    NEXT @i
]
```

### Example 2: Iterating Through a Rowset (Advanced Scenario)

This example shows how to use a `FOR` loop to iterate through a rowset created from a string, demonstrating how to process multiple items dynamically.

```html
%%[
    /* Define variables */
    VAR @rowset, @rowCount, @i, @item

    /* Create a rowset from a comma-separated string */
    SET @rowset = BuildRowsetFromString("Apple,Banana,Orange,Grape", ",")
    SET @rowCount = RowCount(@rowset)

    /* Check if the rowset is not empty before looping */
    IF NOT EMPTY(@rowset) THEN
        /* Loop through each row in the rowset */
        FOR @i = 1 TO @rowCount DO
            SET @item = Field(Row(@rowset, @i), 1)
]
            <p>Item %%=v(@i)=%%: %%=v(@item)=%%</p>
%%[
        NEXT @i
    ELSE
]
        <p>No items to display.</p>
%%[
    ENDIF
]
```

### Example 3: Decrementing Loop

This example demonstrates a `FOR` loop that counts down from 5 to 1 using `DOWNTO`.

```html
%%[
    /* Define the counter variable */
    VAR @j

    /* Loop from 5 down to 1 */
    FOR @j = 5 DOWNTO 1 DO
]
        <p>Countdown: %%=v(@j)=%%</p>
%%[
    NEXT @j
]
```
