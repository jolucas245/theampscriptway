title: "InsertData"
---


The `InsertData` function is used in Salesforce Marketing Cloud AMPscript to insert a new row into a specified data extension. This function is particularly useful for capturing data from CloudPages, landing pages, microsites, and SMS messages. It returns the number of rows successfully inserted, which is typically `1` upon successful execution.

### Example 2: Advanced Scenario - Form Submission with Validation

This example demonstrates how to handle a form submission on a CloudPage, validate the input, and then insert the data into a Data Extension. It includes defensive coding practices to ensure data integrity.

```html
%%[
/* Declare variables to hold form input values */
VAR @firstName, @lastName, @email, @consent, @insertResult

/* Retrieve form input values using RequestParameter. */
/* Trim whitespace and set default empty string if parameter is not present. */
SET @firstName = ProperCase(Trim(RequestParameter('firstName')))
SET @lastName = ProperCase(Trim(RequestParameter('lastName')))
SET @email = LowerCase(Trim(RequestParameter('email')))
SET @consent = RequestParameter('consent')

/* Basic validation: Check if required fields are not empty */
IF NOT EMPTY(@firstName) AND NOT EMPTY(@lastName) AND NOT EMPTY(@email) THEN

    /* Check if the email format is valid (a simple check) */
    IF IndexOf(@email, '@') > 0 AND IndexOf(@email, '.') > 0 THEN

        /* Insert data into the 'Subscriber_Form_Data' Data Extension */
        /* Ensure column names match your Data Extension exactly */
        SET @insertResult = InsertData(
            'Subscriber_Form_Data', /* Name of your Data Extension */
            'FirstName', @firstName,
            'LastName', @lastName,
            'EmailAddress', @email,
            'HasConsented', IIF(NOT EMPTY(@consent), 'True', 'False') /* Convert checkbox to boolean string */
        )

        /* Check if the insertion was successful */
        IF @insertResult == 1 THEN
            /* Success message */
            Output(Concat('Data for ', @firstName, ' ', @lastName, ' successfully saved!'))
        ELSE
            /* Error message if insertion failed */
            Output(Concat('Error: Could not save data for ', @email, '. Please try again.'))
        ENDIF

    ELSE
        /* Error message for invalid email format */
        Output('Error: Please provide a valid email address.')
    ENDIF

ELSE
    /* Error message for missing required fields */
    Output('Error: All required fields (First Name, Last Name, Email) must be filled.')
ENDIF
]%%
```
