## Overview

AMPscript includes several functions for working with website or ‘site-based’ content in Marketing Cloud. See the table below for a few different options for creating site-based content in Marketing Cloud:

| Application | Feature | Description |
| --- | --- | --- |
| Email Studio | Landing Pages (Classic Content) | Stand-alone, individual web pages that can be published with static or personalized content |
| Email Studio | Microsites (Classic Content) | A collection of one or more landing pages |
| CloudPages | Landing Pages | Stand-alone, individual pages that can be published with static or personalized content with editor interface options (Classic Editor or Content Builder) |
| CloudPages | Code Resource | JavaScript, CSS, JSON or other supporting web page assets |
| CloudPages | Microsites | A collection of one or more landing pages with a graphical site map interface |

Applications for AMPscript and site-based content include:

* Personalized landing pages for marketing campaigns, promotions or offers
* Custom preference centers
* Web pages and forms that integrate with data from Sales and Service Cloud Objects

> TIP: AMPscript can be used in CloudPage Code Resources; for example, to conditionally change CSS styles based on the Business Unit MID, or to modify styles based on URL parameters.
